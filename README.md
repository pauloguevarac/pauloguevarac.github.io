## 💻 Running Local Site Development

If you want to run this Jekyll blog locally, follow these steps:

### Prerequisites
Make sure you have Ruby and Bundler installed.

### Setup & Launch
1. Clone this repository:
   ```bash
   git clone https://github.com/pauloguevarac/pauloguevarac.github.io.git
   cd pauloguevarac.github.io
   ```
2. Install dependencies:
   ```bash
   bundle install
   ```
3. Start the local server:
   ```bash
   bundle exec jekyll serve
   ```
4. Open your browser and navigate to `http://localhost:4000`.
